Блок навигации

  • Ширина 15em
  • Высота определяется контентом
  • Должен присутствовать цвет фона
  • Рвсполагается слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Высота по размеру контента
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей равным 2 ем, при помощи свойства margin-bottom

Статья 2

Размер блока
занимает всю доступную ширину
Высота по размеру контента
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей равным 2ем,при помощи свойства margin-bottom. Картинка вставлена про помощи тега, плавание организовано по правому краю внутри данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
