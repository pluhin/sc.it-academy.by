Блок навигации
  • Ширина 15em
  • Должен присутствоать цвет фона
  • Располагается слева
  • Блок навигации наполните контектом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показанно в данном примере
Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутсвовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным .2em, при помощи свойства margin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутсвовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным .2em, при помощи свойства margin-bottom. Картинка вставленна при помощи тега,плавание организованно по правому краю внутри данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещённым внутрь контентом.
